Kpop Guessing Game

Database Initializer & Handler

This is the database initializer and handler for the Kpop Guessing Game PHP and AJAX iterations. This program does the following:

  • Establishing a connection to the local server.
  • Verifying if the given database has already been instantiated and initializes it if not.
  • Closing the connection made to the local server.
  • Displaying all table entries for the 'groups' table.
